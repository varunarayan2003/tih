# 🎓 Student Portal (Streamlit + SQLite)

A simple and interactive **Student Portal Web Application** built using **Streamlit** and **SQLite**.
It allows students to log in and view their **semester-wise marks**, **SGPA**, and **CGPA** in a clean dashboard.

---

## 🚀 Features

* 🔐 Student Login System
* 👤 Student Profile with Photo
* 📚 Semester-wise Marks View
* 📊 SGPA & CGPA Calculation
* 🗄 SQLite Database Integration
* ⚡ Fast and Lightweight UI (Streamlit)

---

## 🛠 Tech Stack

* Python
* Streamlit
* SQLite
* Pandas

## ▶️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/student-portal.git
cd student-portal
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the app

```
streamlit run app.py
```
